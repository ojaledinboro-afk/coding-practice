# Coding Practice Repository

This repository contains small projects organized by programming language and technology.

## Java Projects
- **Array Processing** – Demonstrates working with arrays and basic algorithms.  
- **Baseball Stats** – Processes baseball statistics and performs calculations.

## Python Projects
- **Choose Your Own Adventure** – Text-based adventure game with branching paths.  
- **Number Guesser** – A game where the computer guesses or the player guesses numbers.  
- **Pig** – Multi-player dice game with turn-based scoring.  
- **Quiz Game** – Command-line quiz game with questions and scoring.  
- **Rock Paper Scissors** – Player vs computer game with score tracking.  
- **Timed Math Questions** – Math quiz with time limits per question.  
- **User Input Storage** – Stores and retrieves user input data using Python.
