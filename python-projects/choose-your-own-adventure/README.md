# Apocalypse Adventure Game

Welcome to **Apocalypse Adventure**, a text-based choose-your-own-adventure game where you play as a kid trying to survive in a post-apocalyptic world. Make smart choices to survive—every decision matters!

---

## Features

- **Interactive story:** Navigate through streets, alleys, and ruined buildings.  
- **Multiple endings:** Your choices determine if you survive or get caught in danger.  
- **Expandable engine:** Easy to add new story paths, characters, and locations.  
- **Input validation:** The game only accepts valid options, so you can focus on strategy.

---

## How to Play

1. Clone or download this repository.  
2. Run the game using Python 3:

```bash
python choose_your_own_adventure.py
