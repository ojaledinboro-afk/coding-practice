# Name Manager (Prev: User Input Storage)

A simple and interactive Python program to **store, view, search, and manage a list of names** with persistent storage. Perfect for keeping track of names between program runs.  

---

## Features

- **Add Names** – Input multiple names with the option to quit at any time.  
- **View Names** – Display the full list of names with numbering.  
- **Remove Names** – Delete a specific name by number.  
- **Search Names** – Find names by keyword (case-insensitive).  
- **Persistent Storage** – Names are saved in `names.txt` and loaded automatically on startup.  
- **Clean Interactive Menu** – Easy-to-use numbered menu for all actions.  

---

## Requirements

- Python 3.x  
- No external libraries required  

---

## How to Run

1. Clone or download the repository.  
2. Make sure `names.txt` is in the same folder (it will be created automatically if missing).  
3. Run the program:

```bash
python name_manager.py
