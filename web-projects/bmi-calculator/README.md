\# BMI Calculator



A simple and interactive Body Mass Index (BMI) calculator built with HTML, CSS, and JavaScript.

This app allows users to calculate their BMI using either Metric or US (Imperial) units without reloading the page.



\---



\## Features



\* Toggle between Metric and US units

\* Dynamic calculation using the correct formula

\* Input validation for user-friendly error handling

\* Displays BMI category (Underweight, Normal, Overweight, Obese)

\* Clean UI with responsive layout

\* Output box appears only after calculation



\---



\## Formulas Used



Metric System

BMI = weight (kg) / height (m)^2



US System

BMI = (weight (lbs) / height (in)^2) × 703



\---



\## Technologies



\* HTML5

\* CSS3 (Flexbox)

\* JavaScript (DOM Manipulation)



\---



\## How It Works



\* Users enter height and weight

\* Select their preferred unit system (Metric or US)

\* Click Calculate

\* The app validates inputs, computes BMI, and displays the result with a category



\---



\## Example



```

Input:

Weight: 70 kg

Height: 1.75 m



Output:

22.86 (Normal)

```
